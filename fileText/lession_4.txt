JSX là 1 phần mở rộng cú pháp để có thể viết HTML ngay bên trong file JS
Viết chung js và html vào 1 nơi, gọi là components - thành phần

JSX và React là 2 thứ riêng biệt
cần bọc nhiều thẻ html vào 1 thẻ root, như div
hoặc dùng react fragment <> </>
nếu dùng biến trong jsx thì cần bọc trong {}
cú pháp {{}} là của 1 object viết bằng jsx
